# Git HOL 1

## Objective
Learn basic Git commands such as initialization, configuration, adding files, committing changes, and checking repository status.

## Commands Executed

```bash
git --version
git config --global user.name "Aishwarya S"
git config --global user.email "aishwaryasuresh636@gmail.com"
git init
echo Welcome to the version control > welcome.txt
git status
git add welcome.txt
git commit -m "Added welcome.txt"
git status
```

## Result

Successfully configured Git, initialized a repository, created `welcome.txt`, added it to Git, committed the changes, and verified the repository status.
