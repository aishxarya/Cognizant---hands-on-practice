# Git HOL 5 - Cleanup and Push to Remote Repository

## Objective
Learn how to clean up the repository and synchronize local changes with the remote Git repository.

## Tasks Performed

1. Verified that the main branch was clean.
2. Listed all available branches.
3. Pulled the latest changes from the remote repository.
4. Pushed local commits to the remote repository.
5. Verified that the changes were reflected on GitHub.

## Git Commands Used

```bash
git status
git branch
git pull origin main
git push origin main
```

## Result

Successfully synchronized the local repository with the remote GitHub repository and verified that all changes were uploaded successfully.
